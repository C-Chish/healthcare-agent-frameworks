# 🏥 Clinical AI Case Studies

This directory documents real-world problem-solving using Agentic AI within the New Zealand healthcare context. Each case study demonstrates a specific challenge, the technical strategy used to solve it, and the measurable clinical impact.

## 📋 Methodology: The "CARE" Framework
Every project in this folder followsl a rigorous development process to ensure safety and accuracy:
* **Context:** Identifying specific clinical pain points and business challenges.
* **Ask:** Designing specialized prompts with clear personas and objectives.
* **Rules:** Implementing strict constraints, such as HIPC 2020 compliance and NHI redaction.
* **Examples:** Utilizing few-shot prompting to guide the agent toward professional medical outputs.
